***** Welcome to BabyRogue's level editor instructions! *****

It is VERY IMPORTANT that your level respects the following rules if you want the game to accept it without crashing.
It is recommended you use the "sandbox.txt" template file to create your level. Make a copy and get started!
Simply replace any of the characters in the file by your level.

Dimensions: 75 x 17 tiles.

# = wall
. = empty
@ = player
$ = shop
X = chest / treasure
Y = exit level (next)
Z = exit level (previous)
[ANY OTHER LETTER] = monsters
